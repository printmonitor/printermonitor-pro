# PrinterMonitor Pro

Cloud-enabled 3D printer monitoring platform.

**Status:** Phase 0 - Initial Development

## Components

- **proxy/** - Device software for local printer discovery and cloud sync
- **server/** - Cloud API backend for data storage and user management  
- **web/** - Web dashboard for monitoring and management
- **docs/** - Technical documentation
- **infrastructure/** - Deployment configurations

## Development

See individual component READMEs for setup instructions.

## Relationship to PrinterMonitor Community

PrinterMonitor Pro is the commercial cloud-enabled version of [PrinterMonitor](https://github.com/madwonko/printermonitor).

- **PrinterMonitor Community**: Free, open-source, local network only
- **PrinterMonitor Pro**: Commercial, cloud-enabled, remote access
